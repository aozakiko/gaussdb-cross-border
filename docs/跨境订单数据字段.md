# `records` 表字段说明

| 字段 | 数据类型 | 示例 | 说明 |
| --- | --- | --- | --- |
| `id` | SERIAL / INTEGER | `1` | 主键，自增 |
| `title` | VARCHAR(120) | `智能筋膜枪` | 商品或方案名称，用于前端展示 |
| `description` | TEXT | `US · Amazon US` | 备注信息，可写渠道+站点+SLA |
| `status` | VARCHAR(32) | `in_transit` | 履约/物流状态，取值见下方状态表 |
| `priority` | SMALLINT | `3` | 风险/关注等级（1-5），用于排序与筛查 |
| `owner` | VARCHAR(64) | `ops-us` | 负责该订单的跨境运营/客服 |
| `order_number` | VARCHAR(64) UNIQUE | `ORD-2025-0001` | 业务唯一订单号，同一渠道不可重复 |
| `sales_channel` | VARCHAR(64) | `Amazon US` | 订单来源渠道（Amazon、Shopee、自营等） |
| `destination_market` | VARCHAR(64) | `US` | 目的市场 / 国家 / 关区 |
| `currency` | VARCHAR(8) | `USD` | 订单币种，方便后续换算 |
| `order_amount` | NUMERIC(12,2) | `199.99` | 订单金额（含税/不含税均可，通过备注说明） |
| `tracking_number` | VARCHAR(64) | `PKG00001US` | 物流单号，可为空 |
| `order_date` | TIMESTAMPTZ | `2025-01-12 08:00:00+00` | 下单时间，用于按日趋势分析 |
| `created_at` | TIMESTAMPTZ | 自动 | 记录创建时间（应用生成） |
| `updated_at` | TIMESTAMPTZ | 自动 | 最近更新时间（应用/数据库触发） |

## 状态枚举

| 值 | 文案 | 含义 |
| --- | --- | --- |
| `awaiting_fulfillment` | 待履约 | 订单已生成，待仓库/供应商确认 |
| `picking` | 拣货中 | 仓库已接单、正在备货 |
| `in_transit` | 跨境在途 | 已发往目的国，处于跨境运输阶段 |
| `delivered` | 已签收 | 客户已签收，或本地派送完成 |
| `exception` | 异常 | 包括海关滞留、库存缺货、客户拒签等需人工处理的状态 |

> 以上字段已在 `backend/app/models.py`、`schemas.py`、`frontend/` 与 `docs/` 中保持一致，若需扩展 SKU/仓库维度，可在 `models.Record` 中增加字段并同步更新数据字典。
